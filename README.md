# vue2-todolist

>vue 2 个人练手项目

## 使用

### 安装

``` bash
npm install
```
>国内推荐使用 **[cnpm](https://npm.taobao.org/)** 安装

### 运行项目

```bash
npm run dev
```
>打开 [localhost:8080](http://localhost:8080)即可访问项目

### 打包

```bash
# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
